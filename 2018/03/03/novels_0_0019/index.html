保定吸血鬼

2018-03-03
|
  • 短篇小说
访问

求过利剑，来赢得我的自由
求过毒药，来拯救我的怯懦
——波德莱尔《吸血鬼》

　　雄伟壮观的保定中心血站在我们学校，也就是华北电力大学二校区的对面，中间隔着一条街。那一条长街越往西去就越窄，到尽头是旅馆，小吃街和棚户。有一回有家卖黄焖鸡的煤气罐炸了，炸得那头七零八落。但是很离奇，那次一个人也没炸死，有个卖烤红薯的当时就站在煤气罐旁边，煤气罐都发射到天上去了，他居然毫发无伤。后来我见着他的时候老给我说这事。反正这些我是不大懂，后来那一片就拆了。那是夏天，五月间的事情了。校区园子里的乔木笔挺，碧绿深沉的树叶拥在那个洁白的跳伞塔四周，跳伞塔又对着锅炉房的大烟囱，一切向着天空。我们汗流浃背地坐在足球场上遥望，海子有首诗写的就是这里。我去北京实习又回来，回学校和余凡吃了一顿椒麻鸡。白的啤的都喝了点，晚上过了十一点回不去，睡在那头一家叫罗曼酒店的破旅馆里，我们打开电视，看了一会一个叫《男人的全球购物》的真人秀节目。大概讲一个有钱有闲的男人到全世界各地买点工艺品倒卖的经过，我对他说，我有个堂哥在美国学罗曼语专业，这能干嘛。他说，波德莱尔。我想，得，波德莱尔。无聊。我不由沉沉睡去。

　　醒来的时候是冬天了，坐在床边，我发现旅馆地暖打开，房间里弥漫着怪味。余凡躺在那边床上，大衣挂起来，电视上还是全球购物的男人。我拿电热壶烧上水，凑到那个小得像杂志版面的塑钢窗户边，轻轻撩开窗帘。下雪了，当然，这幅人间奇景我们都需要经历。雪落在一片平房的屋顶，稀稀疏疏地间着砖瓦的颜色，那些细屑还在生长。看久了，我怕自己像结晶一样冻住，索性放下窗帘。总之我们稀里糊涂地过了一晚，也只给了一晚的钱，倒是挺划算。这算是冬天的开始。

　　回想起来，入冬以后我们什么也没做。供暖之后，雾霾一度非常严重，先是烟囱和高塔不见，再是楼房隐没，最后是人的失踪。夜幕降临，我提着热水瓶子，在校园里晕头转向。余凡有时候会套着一件黑色的大衣，往学校门口的网吧里跑。而我一直想不起自己该干什么，天气愈冷，我的情绪没有由来地变坏，家中也开始有不好的消息传来。过了圣诞节以后，我大部分的时间都是躺着，我感到我的身上有什么疾病复发了。但我拿不准那是什么病。12月27日和12月28日我一天都在做梦。12月29日和12月30日发了烧。12月31日，外边的世界比往常要高兴，树上也拉了彩灯。我就找到余凡，一起去游园会和文艺汇演凑热闹。天黑以后我们拐进二餐厅的一楼，那地方像个纸盒子一样方方正正，各个院系的同学将乏善可陈但应时应景的设置摆在两边。人们挤挤挨挨，很笨拙地拥到一处，另一处。游园会的时候，这里的好处是很温暖，大家谈话的声音回响，热切都飘在天花板下面，像云一样。

　　我们来回走了两趟，什么也没干，也不快活。我觉得这样不太好，我问我们该干嘛，是玩磁铁走迷宫还是接电路。他说，猜灯谜。我觉得这是我人生中迄今为止听过的最缺乏思考的建议，乏味得就像在冰箱里头宿居六年的——我。但是，为什么不呢。我们不自觉地站在灯谜那个行列的末尾。在地板上跺脚两次之后就换到了前头。桌子后面站着一个短头发的女孩子，穿着浅蓝的毛衣，抱着手，眼睛细长而且游移不定地看着别的地方。我把桌子叩了叩，我说你好呀，我们想猜灯谜。她回过神来，稍纵即逝地笑了笑，弯腰把一个糊了红纸的箱子搬上来。我伸手从里面摸出一个纸卷展开了，上面写：“出淤泥而不染（打一地名，秋千格）”我问她什么是秋千格。她说：“倒过来。”我答花莲。她点点头，那小剪刀从一卷票券上利落地剪下两张来。我们得了奖券，神情笃定，火速窜出二餐厅一层，沿着外面的钢架楼梯上了三楼。这里面每张桌子上都有人下棋，阵仗仿佛围棋定段赛。我们拨开那些看棋不语的君子直达兑奖的柜台，那里早上还在卖面点，现在俨然是国营百货商店。那些个同学戴着眼睛，领子很高，手底下的所有货都出自义乌。我到了跟前，目光掠过瓶瓶罐罐，信用卡套之类的物什。最后认定了一种浅粉色的，带LED灯发光的蝴蝶结发卡。这实在是土得要死，比我猜过的一切谜语都要土，土的好像经过多次提纯的，最纯粹的塑料。但盯着它，我有一种直觉，带着它我能交好运，我能多活五十年，我能在雷霆到来之际乘着小船走过四川，在人间流血不止的时候依然继续我的生命。这是爱，我想了想，我爱着，我把你叫爱。我拿出奖券换走了我的护身符，打开灯光，顺手就把爱别在了头上。LED放的是红光，我招摇着走下二餐厅。校园里飘着歌曲，我说走啊，去看文艺汇演。

　　文艺汇演在羽毛球馆里，我们进去的时候人们站着看台上演一个够无聊的小品。看着一阵实在觉得肉麻就又溜出来，一拨人在挂了彩灯的快件寄存柜旁边拍照。我们漫无目的地走得冷透了，又只好跑回表演现场取暖。一来二去，歌舞也换了几个，最后竟然谢幕了。我想起这里接着还要办舞会，就呆着没动。我们兴趣索然地打着哈欠。等他们收拾好了，所谓的舞会大概也就是那么一回事。先是一群人排成行转着圈跳兔子舞，然后又变成各种各样的阵势跳各种各样的，反正看上去挺快活的舞蹈。中间还插上某个系的女孩子的现代舞表演。这里的音乐放得很响，我的内心也有稍稍的撼动。我们有一半的时间是靠在暖气片上面对墙，消磨一年中最后的时光。另一半的时间实在是闲得发慌，就瞧着跳舞的人，在心中评头论足。我的目光游来荡去，开始觉得这些人都有点傻气，就轻轻地笑了出来，继而变成冒犯的放声大笑。这让人十分讨厌，纷纷侧目。最后一个女孩子走过来，问我笑什么。我看过去，恰巧就是刚才游园会上的那位。我多少有些尴尬，我没办法告诉她笑不过是我内心放荡和惶恐的无意识投射。她穿着大衣，双手插在兜里，眼睛往我头顶上瞄，又厌恶地眯了一下。我是真是抱歉，我有点控制不住自己。她看着舞台那边，问我不跳舞吗。我想了想，我说不跳。不跳舞就出去，她简短地说。我哑口无言，注意到这个女孩眼窝很深。余凡出来打个圆场，把我拖出羽毛球馆。我们绕过锅炉房，走到那个跳伞塔下的小园子的时候，天上又开始飘雪。其实一次又一次的，我们都不知道雪是什么开始下的。

　　李密梦见跳伞塔的时间并不确切，他也梦见过用砖头杀人和被猩猩咬伤。他在梦中切实地塑造跳伞塔的形体，理性和情绪也在逐渐归于同一。他发现跳伞塔下那个铁门上的锁不见了，很自然地推开门来沿着狭窄的梯级螺旋向上。跳伞塔里的光都是从高处的开窗漏下来的，有燕巢在高处。等他走到了塔顶的平台，已经什么都忘记。那是北方的一个上午，风声猎猎。他看见了山，像鲸吞一样朝眼前滚来的地平线。我是一些诗歌草稿，李密感到荒唐，北方只有九棵树。

　　那一天保定中心血站在日落之后像中央银行一样高大。大堂里有烧蜡烛的吊灯，可能也没有。我们给步枪装上消音器，把天线项圈戴在脖子上，顶着风过了泥泞的马路。相互之间没有交谈，吸血鬼的故事其实是这样开始的。

　　新年以后，学期结束。我的境况不太好，不得不和余凡去勤工俭学当吸血鬼猎人两个星期，天黑以后就得来守血站。一天给四百，自费保险是一天两百五十，本着被咬倒霉的态度，余凡说他不买。头两天，辅导员带我们坐校车去河北警官学校的实训基地练打靶，在那个沙地上的靶场，打木托的八一杠。我打了两天，基本脱靶。但临傍晚的时候，大家还是在挤在一间朝西的小屋子里签了保证书，那办公桌上盖了玻璃板，玻璃板下压着一些剪报。我们分了枪，领了一个基数的银弹回去了。我在校车上从弹匣里卸出一枚银弹来看，寒光锃亮，但是尖上的光又多少有些暗哑。我再看的时候，车窗外的暮色四合的保定市都细细地映照成一道影子，往子弹的尖上汇集。我想到他们学软件的，有一本有名的技术书叫《人月神话》又叫做《没有银弹》，这本书其实很积极。没有银弹不妨碍我们进步。

　　后来我也时常考虑银弹之有无，因为守夜真的太无聊了。导员临换班前必让我们上交手机。收音机都不让带。一人发个对讲机，两个人巡十几层。我由此疑心吸血鬼猎人其实根本没有作用，我们的最大用处可能是天亮以后看看我们的死状，尤其是尸体上的咬痕来辨别来访吸血鬼的种类。但既然这个冬天，市委通知要狠抓治安建设，科学治妖，协同防魔，铁腕打鬼（后来我才知道其实校党委武装部领着一个连的预备党员，隔壁警官学校的未来干警天天在血站周围埋伏（张俊，后来的全国特警比武狙击项目第七名，当时和他的同学在血站对面电机系学院楼办公室的一个临街的窗户旁边摆了一张行军床，上面放了一套铺盖，每夜轮班驻守）这叫打个诱敌深入，还总结成了经验推广）我们大学生也得响应，所以其实每晚苦熬的时候，意志还算是坚定。

　　凌晨我在无人的洗手间里对着镜子照了照，那副形容是年轻有为的戴了头盔的机关单位保安。余凡在顶层，我打算上去和他汇合。血站二层走廊宽阔又深长，两尽边是各个办公室的门窗，墙上挂着不少宣传资料。走廊灯只开了一半，一明一暗地间隔。恐怖片里死人多的医院就是这个样子，远端楼梯间的地方漆黑，我往那道走的时候心里还是不住发怵。我停下来靠着墙歇会。从大衣里摸出那个夹带的肉夹馍来吃，我知道这样子很不雅观，但那时我活得很狼狈，也满不在乎，这是我的青年时代。那个肉夹馍凉透了，硬得铁石心肠，酱驴肉咸得好像有一股馊味。我硬是咽下去了，拿面巾纸擦了嘴。喝矿泉水，把垃圾都塞到一个不锈钢垃圾桶里。每做一件这样琐碎的事情，我就停顿很久，想着这个长夜还剩下多少。血站走廊的暖气开着，但是若有若无，在半空中，像一只看不见金丝雀一样游离。我跟着金丝雀，跟着我青年时代的最后一口热气，走步梯上楼。我背着枪，抄着手瑟瑟缩缩。我的祖父参加过志愿军，在五次战役的时候经历过很冷的冬天，和那相比我什么也不是。但另一方面，今时不同往日，在2017年，人们生活的方式很不一样，痛苦却有着情感上的一致，而且现代性改变了我们，现代性毁掉了很多的事情，使我们再也不能，再也不能在一种使命感底下过活，这是现代。

　　现代也时常停电。我听见警铃大作又戛然而止，楼梯间应急灯亮起，这通常代表吸血鬼将要来袭，我眯着眼睛。我取下枪上了膛。我在电台里呼叫余凡，我说你到东楼梯间往下走，我往上走，我们尽快汇合。我说，看到人影就喊口令，答不出就打死他。他问口令是啥，我说：问，波德莱尔；答，《人月神话》。他说这两者之间有个球关系，我说口令就是要让人猜不到。他说，行行，保持联系。

　　安静。外面好像起了风，我贴着墙往上走。楼梯间有个窗户，我分了心，往玻璃上伸手摸，冻得我缩回手。外面街灯亮着，我能望到我们学校里面去，学院楼…跳伞塔还在那。我有点害怕了，从窗户跳出去，逃回学校算了。我笑着想，又懊悔自己的分心。我有什么可怕的呢，我擦了擦手心的汗，不过是死。我闭眼力求镇静。

　　啪嚓。我听见玻璃碎裂！我一睁开眼，五月阳光灿烂，小吃街的尽头熙熙攘攘，卖烤红薯的人站在煤气罐边上抽烟。火光，火光，世界越来越热，一切归于终结。不对，不是这个。不对，不是这个。我深呼吸，闭眼，深沉地呼吸。光亮消散，太阳总是会下山的。我再次感觉到阴冷吹动。我再睁眼，玻璃窗外的光亮把我的影子集在楼梯间一角走管道的地方。地上有个面巾纸包装袋是我扔的。不要幻觉，我心想。

　　不要幻觉。玻璃是在楼上碎的，我冷静分析。打开保险。端枪喘着气往上走了一层，在滞重的静寂当中，我的骨节发出种种声响，它们在各个频段共振交谈。不要嘲笑我，我心想，我们有共同的命运。等我站定在三楼的拐角，看见地面上有玻璃碎片，粘着泥和雪的鞋印。我瞧了瞧，没什么特别，像是一双不大的运动鞋。原来外头下雪。也不知道吸血鬼是怎么翻进来的。我看见鞋印往走廊上去，我想了想，我打开无线电，我对余凡说我在吃驴火，要花点时间，你就回楼顶等我，不要动。他不知道嚷嚷了什么，我小声说你他妈真烦，关掉了无线电。

　　我跟着鞋印上走廊。头灯什么也照不见。血站三楼多是行政办公室。我举枪摸索行进。在这么狭小的地方打步枪多半会把自己打死。但我的优点是不爱惜生命，所以当那个影子从我五米外一扇门边窜过的时候，我大喊一声开了火。

　　“波德莱尔！”

　　我打的是连发，枪声沉闷。只是肩托在我肩膀上滑了一下，银弹把吊顶打穿了。我后退两步，调整一下，人不见了。我手臂肌肉紧张。过一会又有动静，窸窸窣窣，我冷静再打。直到听见空仓挂机的声音。地上是乱七八糟的一次性塑料杯在滚。操！这时电力供应忽然恢复，灯光一明一暗亮起。我眨了眨眼睛，吸血鬼站在那里，在血站站长办公室的门口。我上了刺刀。看见吸血鬼戴着棒球帽，穿一身运动服，上衣是白的。我深呼吸。

　　那个煤气罐，当时情况是这样。我不是卖烤地瓜吗，就搁在宇哥他们家门口。宇哥他们家门口有个炉子是烤串的，那天下午换的煤气。就那人来换的时候，让我把炉子熄了。我熄了呀，就等着呗。等得无聊得很，就想过去瞧。我操，那时哥我叼着烟，哈哈哈哈。那逼跑得可快了，我都没反应过来，就轰一下。我操，我晕了，我都不知道怎么回事。哈哈哈哈，你说我是不是死了，我都不知道我是不是死了，我也纳闷这哪能不死呢。反正，回头还是卖烤地瓜呗，宇哥让我赔钱，操。我说死了呀，死人怎么给你赔钱。他还想找人弄我，我能咋办呢，别逼我呀。

我们不过是走投无路。我气血上涌，架着刺刀上去。吸血鬼没躲开，刺刀扎在他左肋。我忽然想起，刺刀不是银的！我气急败坏，大声叫骂，操你妈的！贱种！吸血鬼尽力一拧，帽子掉了。刺刀居然断了下来。一点血色见不着，我看清楚了，衣服上画了个小熊。我冷笑一声，这吸血鬼还是个女流。她开始摇头。我抬头看过去，是我跨年夜上见过的那个女孩。

　　她平静地仰头，说把那个紫外线灯关掉。我关掉回来之后，她说，吸血鬼可以预见多重的世界，吸血鬼在多重的世界间作出选择，吸血鬼用人的血液来延续多重的生命。

　　“？”

　　“就是人择原理。我利用你，这个时间线上，这个地方的你不会死。所以我让你杀了他。”

　　“？？”

　　“就好比量子自杀。”她举枪对着我，“我打不死你的时候我就明白了。你想再试试吗？”

　　“不。”我说，“闭嘴吧，我听不懂。”

　　她悻悻地抱着手臂，站起来。巡视了一下血库，她说可惜这里已经没有血了。她说，血都调到北京去了。

　　我说我还以为你知道，她说什么意思。我说，你不是吸血鬼猎人吗，你肯定也是在学校报了名的，你他妈根本就是走正门刷卡进来的。我推理道，那个卖红薯才是从三楼破窗而入，直奔血库去了。我又顿了顿，你他妈早就知道血库有人。

　　她微微笑，拍手。“反正现在人也被你杀了。”我说是你欠我的，你要帮我个忙。她想了想，让我说。我说，咬死我，在这咬死我。她说，为什么？

　　我说你需要血，我需要钱，家里需要钱，我买了保险，我还杀了吸血鬼，如果我死了保险和抚恤金够他们花的了。

　　“你活着也能挣奖金呀。”

　　“不想活呗。”

　　“真的？”

　　我闭上眼睛：活着并不是没有意义的感觉，总的来说，随着年龄的增长呈现出一种限量供应的态势。当然，在这方面不应当有过高的奢求，我是认可的。有些事情已经过去，有人叫我的名字，我时有焦灼，一跛一跛地走在屋檐下，地上，没水的游泳池底，随便什么地方。啊，是的，我试探地说，这不就是——就是我吗，那一年针叶树下的土丘都像是我的坟冢，把门背拍响的人也是我。一段时间里，太阳和空气就像松掉的螺丝钉那样掉落，在这个间隙里面被弄混。邻居上班走过死鹿，这一天勤勤恳恳，让我羡慕，接着另外的一天，让我羡慕。我只有注视，我在注视，有没有声音是正在鸣响的，我不知道。当一切像一面正在干涸的钟，我不知道人能不能只守着空围栏，然后就这样活下来。

　　我说真的，她毫不客气地伸出左手把我按到柜子上去，张开嘴。我说等等，我从兜里掏出那个LED灯的发夹，递给她。她说这是啥，我迟疑了一下，还是告诉她这是“爱”，从你那弄来的。她哑然失笑，我耸肩，我郑重其事地说花莲，留作纪念。她看着我，缓缓把发夹别上。再用一个很柔和的动作靠过来，往我颈动脉上咬了一口。我好似失声一样掉进深井，这一切发生得这么快，我想，她真的这样做了，我有些吃惊，但不失望，手臂举不动了，头发挠得我脖子发痒，没有疼痛，我感到困倦，我轻声说项圈的事情是编出来骗你的，她说她知道，她的眼睛，月……

　　波德莱尔我说《恶之花》。不顾一切感到黑暗沉默和完整 关于地狱我将会在以后的岁月里向你们一一叙述底层的神的灵感和灵魂的深层涌泉代表着被覆盖的秘密的泉源形体死灭浮生便若梦幻如未就的画稿中途搁浅面对遗忘的画布无从下笔画家只能徒将记忆呼唤倾向于用更简单的概念来描述复杂事物现在你可能是正确的我无法断定所有的复杂事物都容易用有序的规律表达绝对命运默示录黑暗沙漠里的灿场宇叶是镀了金的桃花源白天与黑夜逆转之处是镀了时间的失乐园。波德莱尔我说《恶之花》。

　　我被枪声吓醒了，我痛得要命，像雕像一样从亘古的石块里面一凿一凿地浮现出来。我硬挺挺地倒着，睁眼还是血库的天花板，傻逼余凡抱着枪来摇我，他说他妈都天亮了，这都发生了什么。我说，你，他妈，开枪，干嘛！他说你别乱答口令啊，我说口令是啥。他说，《银月神话》。我闷闷地说，这他妈有个屁的关系。他说，口令就是要让人猜不到。我无语，我说你别鸡巴废话了，我花了几分钟坐起来，忽然想到，急忙撩开衣领给他看脖子，我说，这里有伤口吗。他对着光看了好一会，他说没有。我愕然，我又让他再看，他仔仔细细地看过之后，他说没有。

　　就这样，我活着，也有很多弄不明白的事情。我们一直在血库里待到天亮，我想着花莲已经走远，就报了警。我立了功，拿了一点钱，离开保定之后，人生一泻千里，父亲病了。我的病有时好了，有时不见好，2020年我去了上海，和余凡在闸北区住了一年。而后返乡，将近十五年未曾北上。等到2035年我因公事往北京，在酒店约炮，和一个身材敦实，精力充沛的女人做爱，生命在这个阶段很松散地散失，我什么也不是。她的眉毛画得很深，像一桩要事那样沉重，我有责任，这期间惶恐地敲打牙齿。她对我说，李密，你没意思。她把裤袜往大腿上套，我伸手抚过她腿上的毛孔和鸡皮疙瘩，我说这天气是有点冷的。我另一手按遥控器，把电视上的频道都浏览了个遍，好嘛，男人的电视购物再也没有了。她把毛衣裙穿好，我在穿衣镜前从背后环抱着她。我说今天我和客户喝酒的时候见到了一个人，她说咋啦，我说他原来是武警的狙击手，后来转业到地方，搞安保。她说你认识他？我说第一次见，他说他叫张俊，以前在河北警官学院。她说，是你同学？我说，我以前也在保定呆过——他今天喝多了说，有一回，上头下命令，让他们小组把夜晚从血站出来的人全打死。她说为啥啊，打坏人吗，我摇头说，那次他在瞄准镜下看见一个女孩子，头顶戴着一个发红光的东西。她说，怎么回事？我说，张俊也不明白，他心里一动，准星跳动半分。

　　我拿手比作手枪，抵着她的头：砰！然后我说我爱她，理所当然。
