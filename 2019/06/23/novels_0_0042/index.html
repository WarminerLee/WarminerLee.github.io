“我？”Wonderkey轻轻地笑着说，“我早就知道了——我试图寻找自由意志让我更加接受自由意志不存在的事实，也就接受着人生无法真正自由选择的人生观，我只能做着我要做的事。”

“你真的知道？”你吃惊地说，“我没开玩笑啊，你真的是一个游戏里的NPC！”

“直觉。”他说，“有种直觉。”

你在想他和所有你见过的数学家都不一样——数学和数学以外的东西在构造他的时候占据了同等分量。

“如果我没猜错的话。”他说，“在你的所谓‘真实’的宇宙中，一定有一个我的原型人物吧。”

“……有、有的。”

“他过着怎么样的生活呢？”

“他是一个偶像。”你信口开河地说道，“是一个很受欢迎的唱跳型选手。”

“真的假的？”

“真的。”你想，反正平行宇宙有无穷个。

“嘿嘿。”Wonderkey摸着头说，“我这样的人也可以成为偶像吗？”

“可以。”你说，“不行就多试几次。”

Wonderkey轻轻摇头，“对于我来说这不是一个游戏。”

他是对的，你说不出话来。悲伤充满了你的心灵，人和人生来就是不一样的，你永远无法成为天才，可是他却被创造在一个毫无意义的世界里毫无意义地一次次被毁灭。你恨这个粪游戏的制作者，你想把光碟掰了。

他看了看表，语气轻松地说，“虽然还有两分钟了，不过，你肯定肯定还可以见到我——另一个我。所以，我也不必说得太郑重，对吧。”

“这一切都不会是毫无意义的！”你说出了系统安排好的过场台词，“这世界上的一切都是有答案的！总有一天…”

Wonderkey第一次显现出迷茫，他不信任你的话，你觉得自己像个傻瓜，于是你也说不下去了。接着，他眼睛一亮，“你想听我唱歌吗？”

“唱唱唱。”你说，“抓紧时间，不然我们就要爆炸了！”

Wonderkey低下了头，“算了……有点，有点不好意思。”

你再也受不了这样悲伤的对白了，反正过了存档点了，你想立即去世，你不想被炸弹炸成烟花，你点开Wonderkey的题目，你没看懂，你手指放在随便一个选项上。

你抬起头，Wonderkey也看着这一切，你们沉默着对视了一会，你把点头作为告别，你按下按钮，还好选错了，毒药正在注射进你的身体，平静的感受从血管流遍全身。你闭上眼睛倒在地上。

你死去了。

你听到有轻轻的歌声响起。

聚变炸弹爆炸，把一切都变成了光。

和一个集合不同，你永远也不知道这三件事情的发生顺序。
