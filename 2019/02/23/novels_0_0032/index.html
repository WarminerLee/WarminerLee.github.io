轻小说草稿：超级魔法师大战（其一）

2019-02-23
|
  • 短篇小说
访问

　　太久不更新，混混，写着玩，有空就编一点。

片段1

　　“你如果输掉超级魔法师大战。”列车长对我说，“狗熊人就会统治这个世界。”

　　我刚刚发觉这个夜晚，这趟只载我一个乘客的火车可能已经早就开过了两个世界的交界。你看到我这么说，肯定会想起哈利波特，我也喜欢哈利波特的故事，可我没有把自己的经历和哈利波特相比，我们之间有很多不同。

　　“你们真的很在乎吗？”我说。

　　列车长撑着卧铺车厢床边的桌板，扶正帽子说了一通。我那个时候正好走神了，所以我无法复述他的话。那个时候火车正开过青海湖，月亮高照，那景象极为动人。但是这个景象也不应该被描述，我将把不描述这个景象的道理讲通，这其中的原理涉及了世界的本质和活着的意义，我有一半是在瞎掰，但当我望向湖面的时候，我当时就意识到了这个时刻意义重大，我应该照张相什么的做个纪念，但我没有——因为已经不需要了，月亮和青海湖，隔着模糊的玻璃，我领悟到两件事，一来，转瞬和永恒是可以统一的时间性质，你可能已经知道，二来，我无法记住任何事，这是我配不上的，这份使命生来只属于月亮和青海湖。所以我便什么也不说，维持着不值一提的沉默，我低下头，列车长说完了他的话。转过身去，回到了乘务员室。没人管我，我动也不想动，就干巴巴地坐着，下颌向内收。我的脸已经很靠近窗玻璃的时候，我就打个激灵，掏出没有信号的华为手机，点开微信看他们对我说过的话。

片段2

　　我导师左手手心向上摊在桌面，手掌里燃烧着火焰，焰色忽而成浅蓝，忽而又偏向不易察觉的新绿色，火苗向上挺直。我们的工坊本来就亮着几条日光灯管（看起来又脏又暗），但他点起火来，好像意味着要和我说一些郑重而不合时宜的话，不过滑稽的是，他右手在兜里摸来捻去，把瘪掉的烟盒，钥匙串和小票扔到桌面上，把那个烟盒倒过来敲，又艰难地用右手试图去够着左边的口袋。我站起来，去抽屉底下的一条里拆了一盒烟回来，捻出一条来在他手心点燃，恭敬地交给他。他一吐烟，我很难掩饰地轻轻皱眉抽鼻子，他说，小强，你坐，你坐。

　　我坐在工坊绿色实验台的一边窄窄的过道，背后是文件柜，脚边有一卷软管，一个塑料凳子上放着一个塑料容器盒。桌子上是插座，试管架子，计算器。我局促不安地挺直背靠在文件柜上。导师双手缓慢地在桌上合在一起，啪的一下，火焰熄灭。我在火焰熄灭的前一刻说：

　　“我去找了张诚——他跟我说系里面的意思就是说，我要去，没问题，原则上不鼓励也不反对，学工处那边我也把知情书和资质证明那些材料给他们盖章了，现在就是说主要等你开介绍信。”

　　“退学呢？也办了？”

　　“那个院里面说好弄，叫我最后一天去就行了。退宿那些也是。”

　　“哦。”我导师李博文眉头伸展，掐着烟老半天不说话，找了个研钵把烟摁了。去水池边上打开杯子喝了点茶，顺手打开了通风扇，他背对我说：“你觉得你自己聪明吗？”

　　“呵呵，我就一般吧，和老梁那样的大牛还是差距挺大的，不过小聪明还是有点，看问题比较灵活吧。”

　　“你学魔法花功夫了吗？”

　　“还行吧，就还行，我自己爱好就在这一块，还是比较投入的，就是有时候，哈哈，怎么说，心有余而力不足。”

　　他转过来，双手握着杯子，微微点头：“你态度我觉得还是可以的，，热情够，魔法学习背景这块也比较宽，还有就是我觉得你团队感啊，为人这些都不错——你就不要把我当导师，你把我当朋友，我也愿意把你当朋友。”

　　“呵呵，没有没有，我就为人很随便的，不懂这些，李教授多包涵。”

　　“我把你当朋友，就不说客套话了，我就老实说，你呢，实力和天赋呢，在我带过的学生里面，属于中下。”

　　我早就准备好了对答：

　　“我觉得宗神塔也不是给什么人都发超级魔法师大战的邀请，而且我知道我自己水平和同学比一般，可是考虑到我们北大也是国内顶尖学校，我不可能连这点自信都没有吧，我的个人发展还是对得起学校，对的起教授的吧，起码。”

　　“小强，我知道你不差——超级魔法师大战不是魔法师大战，你知道什么是超级魔法师——超级魔法师不是靠——小强，你在同龄人里面绝对很优秀，但是超级魔法师，你肯定不是。我也不是，你本科哪的？”

　　“莱阳师范学院。”

　　“我不是说你本科不好就怎么怎么样，我不是说歧视什么，我是说你本科和我的环境不一样，见的东西不一样，对有些东西的认识就有差距——真的，我不是批评你，我把你当朋友我才说，我十六岁北大少年班本科毕业，22岁在钟塔拿了元素和黑魔两个学位，30岁之前赢的国际一线比赛，不比现在院里任何一个教授少吧？我的意思不是说我自己就是天才，我是说，你要相信我，我在这行——我是聪明人，不是蠢人——我比你天赋高多了，你总承认吧。”

　　“哈哈，你不比我厉害，怎么做我老师呢？”

　　他苦涩而语无伦次地说：“我，我不是超级魔法师——我知道你们这个年纪心气高，想挑战，总没错的，但是要选对方法。”

　　“唉，试试吧，我就——”

　　“你就什么？”他说，“我真的跟你讲得很明白了，超级魔法师大战不是魔法师大战，是超级，知道什么叫做超级吗？语文学过吗？超级！你到这一步，到我们北大来，很不容易了，考虑到你的家庭，你的背景。但是超级魔法师这种东西，他就不是人能够，能够说后天塑造的，你这一生，绝对没有办法，我就用我魔法界的资历和荣誉担保，你绝对没有办法进入超级魔法师的行列，但是这没问题呀，可以享受生活，而且我们这个方向还有很多工作要做——”

　　“我有理想要实现。”

　　“啥理想，你跟我干，干好了——”

　　“政治理想。”

　　李博文笑得好难看啊，他咧着嘴又不想出声，干咳然后喝水，左手腾起火苗又一握拳：“你听谁说，不要搞这些虚的东西，你这个人就是不踏实。”

　　“我要赢得超级魔法师大战，然后实现我的理想。”

　　“干什么，演电影啊，动画片啊，我就操了，你这个人傻逼吧，你这波进去必死我跟你说，还政治理想，没人理你的我跟你说，谁理你了？你不小了，27岁还在背那几条咒语，老子27岁在大战一个人杀八个，你以为我跟你吹牛皮？我有什么理想？我就捡了条命回来，真的，我是鬼门关上过来的，我什么没见过，我知道人完得太容易了，一个护盾起不对中道闪电咒就完了，魔法师也是人，肉长的，对吧——超级魔法师就不是人，你想想。”

　　李博文说是导师，其实只比我大三岁，我还在读高中，李博文都纵横江湖了，这我当然知道，他是真把我当朋友，我也知道，可是他是他，我是我。我们的生命是如此不同，从开始一直到结束。不过这没什么好感叹的，人和人就是处处不同的，这不是自明的吗？

　　“反正介绍信你要给到我，不然我自己搞了。”

　　李博文把水龙头打开洗烧焦的手。

片段3

　　此人是我的学生和朋友，系独生子女，家庭经济条件困难，其父11年起因工伤瘫痪在床，其母无固定生活来源。此人好高骛远，眼高手低，在魔法上天赋平平，技巧拙劣。恕本人无法接受宗神塔认定其超级魔法师大战资格的决定。然而事已至此，希望能贵方妥善接收该生。本人强烈要求贵方鉴于其家庭情况予以特殊照顾，务必保证其人身安全。

　　　　　　　　　　　　　　　　　　　　　　　　　　　　北京大学 李博文

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　2023.9.23

片段4

　　站台上，只有那个人坐在长椅上。他穿着冲锋衣，缩着手，我走过去，他就抬起头，那时是下午。他站起来很自然地说，你来啦，路上辛苦吗。我发现他好像个子不高，他接了我的行李箱，右手伸过来，“天津大学，李博文。”我当然本能地有点吃惊。

　　“你叫李博文？”
　　
　　“怎么了？哦，你不是找我吗？可是他们说超级——”
　　
　　“不是，我是来参加超级魔法师大战的，就是我有个，有个朋友也叫李博文，觉得有点意思。”
　　
　　“哈哈，不奇怪不奇怪，叫李博文的人多的是。”
　　
　　“哈哈哈，但是这也太巧了，看来我们有缘啊，我是莱阳师范大学的陈强，一般都叫我小强。”
　　
　　“莱阳师范大学？”李博文说，“真有这学校？”
　　
　　“以前叫莱阳师范学院，后来更名了。”
　　
　　“哦哦哦，这样。”李博文说，“反正——反正我们现在是队友了，有什么事情都可以找我，反正我先到几天，对这边比较熟悉。”
　　
　　“我们组剩下那个同学呢？也到了？”
　　
　　“小卢？她昨天刚来，我和她见过了，她人挺不错的，水平也很高。”
　　
　　李博文拖着我的行李箱走在前头，有一条新修的水泥路从车站去大本营，下午本来很安静，但因为地不平整，行李箱发出很大的声音。
　　
　　“那就好，反正以后我们就互相帮助吧，我可能水平不如你们，但是我能做的肯定尽力。”
　　
　　李博文立住行李箱站住，他回头对我说：
　　
　　“你的愿望是什么？”
　　
　　两边是宿舍楼，前去有一条狭长的通道，李博文就立在当中问我，我看着路边新搞的绿化，我说：“哈哈，刚才我们在聊这个吗？”
　　
　　“没有。”李博文说，“随便问问。”
　　
　　那我就随便答答，我说，“我的愿望是给我爸治病。”
　　
　　“不对。”李博文说，“不是这个。”
　　
　　“啊？什么意思？”
　　
　　李博文说：“我的愿望是永恒的生命，你的愿望是什么。”
　　
　　我身上一冷，“把行李箱还给我。”我说，“不然杀了你。”
　　
　　“好。”李博文一推，行李箱滑过来。
　　
　　李博文肩膀一动，我下意识隔住一道冰墙，左手拔出魔杖，我看见黑暗能量贯穿冰墙，我右踏一步躲闪，闭上眼在杖尖点了个闪光，行李箱我操，我心想，行李箱在哪。我一扭头望见那个绿箱子躺在地上，李博文就从天而降，一道暗影穿刺了我的左腹部，我把血液和内脏冻住，六个水元素拔地而起，站位重重叠叠，3秒之后还剩下一个，它把行李箱举着朝我甩过来，暗影随着也把它吞没了。

　　我接住行李箱，免疫咒语继而生效。屏障维持45秒之后，我的生命就要结束了，我就这样输掉了超级魔法师大战。我凄然地把行李箱冻到零下两百七十度，拿运动鞋一碰它就化为碎屑——

　　李博文隔着屏障，就像青海湖和月亮隔着玻璃一样对我说：“没用的，你看箱子是空的。”

　　我盯着脚下，确实如此，只有行李箱的碎片，这怎么能，这他们怎么能——

　　还没等我想明白，45秒就过去了，心脏就是我时而迅捷时而迟疑的时钟。我一抿嘴，也和行李箱一样化为了灰尘。

片段5

高阶法师刘宇：
　　我的学生陈强如果有意外发生，恳请你协调民政单位，不要就地火化，联系我有关遗体交接事宜。

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　李博文
